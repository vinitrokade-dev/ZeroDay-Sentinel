ZERO DAY DEVELOPERS

External Device password protected system

This project aims to enhance cybersecurity by adding a password-protection mechanism when an external device (USB, hard drive, etc.) is connected to a computer. If the correct password is entered, access is granted; otherwise, access is denied.

Key Features
  • Automatic Detection: Detects any external device (USB, hard drives, etc.) upon insertion.
  • Password Authentication: Requires users to enter a password before accessing the device.
  • Access Control: Grants access only upon correct password entry; denies access otherwise.
  • User-Friendly Interface: Simple and intuitive design for easy use.
Benefits
  • Enhanced Security: Protects sensitive data from unauthorized access.
  • Prevention of Data Theft: Stops unauthorized users from copying files to external devices.

  • Easy Integration: Works seamlessly with existing operating systems.
